# University of Lagos Energy Club - React Application

A modern, professional website for the University of Lagos Energy Club, built with React, TypeScript, and Tailwind CSS.

## 🚀 Tech Stack

- **React 18** - Modern UI library
- **TypeScript** - Type-safe JavaScript
- **Vite** - Fast build tool and dev server
- **Tailwind CSS** - Utility-first CSS framework
- **React Router DOM** - Client-side routing
- **AOS (Animate On Scroll)** - Scroll animations
- **Splide** - Lightweight carousel library

## 📁 Project Structure

```
unilag-energy-react/
├── public/
│   └── assets/           # Static assets (images, fonts, logos)
├── src/
│   ├── components/
│   │   ├── layout/       # Layout components (Header, Footer, Layout)
│   │   └── shared/       # Reusable components (ActionButton)
│   ├── pages/            # Page components (Home, About, Gallery, Contact)
│   ├── types/            # TypeScript type definitions
│   ├── App.tsx           # Main app component with routing
│   ├── main.tsx          # Application entry point
│   └── index.css         # Global styles and Tailwind directives
├── tailwind.config.js    # Tailwind configuration
├── tsconfig.json         # TypeScript configuration
└── vite.config.ts        # Vite configuration
```

## 🛠️ Installation

1. **Clone the repository:**
   ```bash
   git clone <repository-url>
   cd unilag-energy-react
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Start the development server:**
   ```bash
   npm run dev
   ```

4. **Open your browser:**
   Navigate to `http://localhost:5173`

## 📜 Available Scripts

- `npm run dev` - Start development server
- `npm run build` - Build for production
- `npm run preview` - Preview production build locally
- `npm run lint` - Run ESLint to check code quality

## 🎨 Design System

### Colors

- **Primary:** `rgba(233, 194, 2, 1)` - Golden yellow
- **Secondary:** `rgba(32, 32, 32, 1)` - Dark gray
- **Main Text:** `rgba(19, 22, 35, 1)` - Near black
- **Sub Text:** `rgba(85, 85, 85, 1)` - Medium gray
- **Border:** `rgba(244, 244, 244, 1)` - Light gray
- **Section Background:** `rgba(250, 250, 250, 1)` - Off white

### Typography

- **Font Family:** MayoClinicSans (Regular & Bold)
- Custom font files located in `/public/assets/font/`

### Animations

- **Float Animation** - Floating effect for hero images
- **Arrow Animation** - Animated arrow indicators
- **AOS Animations** - Scroll-triggered fade/slide effects

## 📄 Pages

1. **Home** (`/`) - Landing page with hero, membership info, divisions, and testimonials
2. **About** (`/about`) - Club information, mission, vision, and objectives
3. **Gallery** (`/gallery`) - Photo gallery with upload placeholders
4. **Contact** (`/contact`) - Contact form and information
